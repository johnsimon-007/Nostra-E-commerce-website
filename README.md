# 🛍️ Nostra - E-commerce Website

## 📌 Overview

Nostra is a beginner-to-intermediate level frontend e-commerce website built using pure HTML, CSS, and JavaScript. The project focuses on implementing core web development concepts such as DOM manipulation, responsive design, and dynamic product filtering.

---

## 🚀 Features

* 🏠 Responsive Home Page with modern layout
* 🛒 Product Collections Page

  * 🔍 Real-time search functionality
  * 🎯 Category-based filtering (Men/Women)
  * 🧠 Combined search + filter logic
* 📦 Dynamic product rendering using JavaScript
* 🧾 Contact Us form with basic validation
* 🎨 Clean UI with CSS Grid & Flexbox
* 📱 Mobile responsive design

---

## 🧠 Concepts Used

* HTML5 semantic structure
* CSS3 (Flexbox, Grid, Media Queries)
* JavaScript (DOM Manipulation, Events, Array Methods like `filter`)
* Dynamic UI rendering

---

## ⚙️ How It Works

* Products are stored in a JavaScript array
* Displayed dynamically using DOM manipulation
* Search filters products based on name
* Category filter narrows results
* Combined logic ensures accurate filtering
